# Envision × Boeing — WIMKA Small Parts Program

Interactive program site for the Boeing Warehousing, Inventory Management, Kitting and Assembly (WIMKA) Small Parts Program.

## Deploy

This is a static site. Push to GitHub and connect to Vercel — no build step required.

```
boeing-wimka/
├── index.html          # Single-page interactive site
├── images/
│   ├── envision-dallas-dusk.jpg    # Hero background
│   ├── envision-dallas-day.jpg     # Facility daytime
│   ├── envision-dallas-flags.jpg   # Flags / closer background
│   └── envision-dallas-side.jpg    # Side angle
└── README.md
```

## Vercel Setup

1. Push this repo to GitHub
2. Import in Vercel → Framework Preset: **Other**
3. No build command needed
4. Output directory: `.` (root)
5. Deploy
